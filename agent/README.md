# Agent - LangChain FastAPI Backend

The `agent` service is a Python backend built with FastAPI and LangChain.

Its role is to orchestrate AI reasoning for the Historical Battle Analyzer, including:
- Managing conversation and analysis workflows
- Calling language models (e.g., OpenAI, Gemini, Groq)
- Integrating with the `mcp-server` (Wikipedia API wrapper) for historical context

Currently, this folder only contains scaffolding and placeholder files. No real chains, tools, or business logic have been implemented yet.
